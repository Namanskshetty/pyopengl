# pyopengl
### Prerequisite
* Python3
## To install and run:
1. Clone the repo
   ```sh
    git clone https://github.com/Namanskshetty/pyopengl.git
    ```
2. Go to folder
   ```sh
   cd pyopengl
   ```
3. Give the executable permission to bin.sh file
   ```sh
   chmod +x bin.sh
   ```
4. Run the pyhton file
    ```sh
    python3 main.py
    ```
